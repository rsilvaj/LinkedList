## Project Linked List

Implementation of Linked List in Java.

### Implementation guidelines
In order to improve code quality and cohesion, the SWE decided by follow the strict methods/practices:
* Using clean-code as reference
* TDD(Test-driven development)
  * → create the test 
  * → execute the test
  * → expect for error
  * → refactor code
  * → increment the solution
  * → run test suit again
